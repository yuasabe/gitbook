Set `GOPATH`. Edit the `~/.zshrc` file and add following line.

```text
export GOPATH=$HOME/go
```

Source the file.

```text
source ~/.zshrc
```

Follow the Vim-wiki [tutorial](https://github.com/fatih/vim-go/wiki/Tutorial#quick-setup).
